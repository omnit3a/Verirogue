# Controls
- `Arrow keys` Movement
- `>` Enter dungeon
- `<` Leave dungeon
- `f` Set fire
- `s` Scavenge for food
- `S` Check character sheet
- `e` Eat food
- `k` Kick down wall
- `m` View starmap
- `p` Go prone/Toggle swimming
- `q` Drink from lake
- `Q` Drink potion
- `i` View inventory
- `,` Pickup item
- `d` Drop item
- `r` Read scroll
- `?` View controls
