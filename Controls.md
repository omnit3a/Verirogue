# Controls
- `Arrow keys` Movement
- `>` Enter tile
- `<` Leave tile
- `f` Set fire
- `s` Scavenge for food
- `S` Check character sheet
- `e` Eat food
- `k` Kick 
- `m` View starmap
- `M` Medical
	- `c` Cauterize
	- `C` Clean floor
	- `d` Cure disease
- `p` Go prone/Toggle swimming
- `q` Drink from lake
- `Q` Drink potion
- `i` View inventory
- `,` Pickup item
- `d` Drop item
- `D` Describe
- `r` Read scroll
- `C` Sneak
- `?` View controls
